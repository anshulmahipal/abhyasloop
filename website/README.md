# TyariWale Marketing Website

Next.js app for tyariwale.com.

## Vercel deployment

**Required:** In Vercel → Project Settings → General:

1. **Root Directory:** set to `website` (edit → select `website` → Save).
2. **Output Directory:** leave empty (do not set to `website`).
3. **Framework Preset:** Next.js (auto-detected when Root is `website`).

Build and install run from `website/`, and the output (`.next/`) is used automatically.
